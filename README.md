# 💬 Chatify — Real-Time MERN Chat Application

## 📌 Project Overview

**Chatify** is a real-time chat application built using the **MERN stack (MongoDB, Express, React, Node.js)** along with **WebSockets** to enable instant messaging between users.

The application establishes **live server–client connections**, allowing messages to be delivered instantly without refreshing the page. This project demonstrates the implementation of **real-time communication**, **full-stack web development**, and **modern web application architecture**.

Chatify highlights how modern applications maintain persistent connections between the client and server to support features such as live messaging, dynamic updates, and responsive user interaction.

---

## ⚙️ Technologies Used

The project is developed using the following technologies:

* **MongoDB** – NoSQL database used to store user data and messages
* **Express.js** – Backend framework for building REST APIs and handling server logic
* **React.js** – Frontend library used to build a dynamic and responsive user interface
* **Node.js** – Runtime environment used to run the server-side application
* **WebSockets / Socket.io** – Enables real-time bidirectional communication between client and server

---

## 🚀 Key Features

Chatify includes several features that demonstrate real-time web application functionality:

* **Real-Time Messaging** – Messages are delivered instantly using WebSocket connections
* **Live Server–Client Communication** – Persistent connection between client and server
* **Full Stack Architecture** – Integration of frontend, backend, and database
* **Dynamic UI Updates** – Chat interface updates automatically when messages arrive
* **Modern Web Application Design** – Built using industry-standard technologies
